localhost:8085 db_espacios /api/espacios /api/tipo-espacio

Gestión de espacios de estacionamiento y tipos. La disponibilidad es actualizada por ms-accesos en cada entrada/salida vía PATCH.

⚡ Feign: ms-reservas y ms-accesos llaman a PATCH /{id}/disponibilidad para bloquear/liberar espacios.

¿Por qué tocar ms-reservas primero? ms-reservas

ms-reservas solo tiene el endpoint cancelar. El flujo de accesos necesita dos estados que hoy no existen:

  • CONFIRMADA — ms-accesos valida este estado antes de registrar la entrada física.
  • FINALIZADA — ms-accesos lo setea al registrar la salida.

Hay que agregar confirmar() y finalizar() a ms-reservas antes de arrancar ms-accesos.

0. Verificar ConflictException en ms-reservas

Revisa si existe ms-reservas/src/.../exception/ConflictException.java. Si no existe, créala:

ms-reservas/…/exception/ConflictException.java

public class ConflictException extends RuntimeException {

}

Y agrégala al GlobalExceptionHandler de ms-reservas:

public ReservaResponseDTO confirmar(Long id) {
    Reserva reserva = reservasRepository
        .findById(id)
        .orElseThrow(() -> new NotFoundException("Reserva no encontrada"));
    if (reserva.getEstado() != EstadoEnums.CONFIRMADA) {
        reserva.setEstado(EstadoEnums.CONFIRMADA);
    }
    return toDTO(reservasRepository.save(reserva));
}

public ReservaResponseDTO finalizar(Long id) {
    Reserva reserva = reservasRepository
        .findById(id)
        .orElseThrow(() -> new NotFoundException("Reserva no encontrada"));
    reserva.setEstado(EstadoEnums.FINALIZADA);
    return toDTO(reservasRepository.save(reserva));
}

GlobalExceptionHandler.java

exception/GlobalExceptionHandler.java 
package estacionamientos.ms_accesos.exception;

import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.http.converter.HttpMessageNotReadableException;
import org.springframework.web.bind.MethodArgumentNotValidException;
import org.springframework.web.bind.annotation.ExceptionHandler;
import org.springframework.web.bind.annotation.RestControllerAdvice;

import java.time.LocalDateTime;
import java.util.HashMap;
import java.util.Map;

@RestControllerAdvice
public class GlobalExceptionHandler {

    private Map<String, Object> construirRespuesta(HttpStatus status, String mensaje) {
        Map<String, Object> response = new HashMap<>();
        response.put("timestamp", LocalDateTime.now());
        response.put("status", status.value());
        response.put("mensaje", mensaje);
        return response;
    }

    @ExceptionHandler(NotFoundException.class)
    public ResponseEntity<Map<String, Object>> manejarNotFound(NotFoundException ex) {
        return new ResponseEntity<>(construirRespuesta(HttpStatus.NOT_FOUND, ex.getMessage()), HttpStatus.NOT_FOUND);
    }

    @ExceptionHandler(ConflictException.class)
    public ResponseEntity<Map<String, Object>> manejarConflicto(ConflictException ex) {
        return new ResponseEntity<>(construirRespuesta(HttpStatus.CONFLICT, ex.getMessage()), HttpStatus.CONFLICT);
    }

    @ExceptionHandler(MethodArgumentNotValidException.class)
    public ResponseEntity<Map<String, Object>> manejarValidaciones(MethodArgumentNotValidException ex) {
        Map<String, Object> response = construirRespuesta(HttpStatus.BAD_REQUEST, "Error de validación");
        Map<String, String> errores = new HashMap<>();
        ex.getBindingResult().getFieldErrors().forEach(e -> errores.put(e.getField(), e.getDefaultMessage()));
        response.put("errores", errores);
        return new ResponseEntity<>(response, HttpStatus.BAD_REQUEST);
    }

    @ExceptionHandler(HttpMessageNotReadableException.class)
    public ResponseEntity<Map<String, Object>> manejarBodyVacio(HttpMessageNotReadableException ex) {
        return new ResponseEntity<>(
            construirRespuesta(HttpStatus.BAD_REQUEST, "El cuerpo de la petición no puede estar vacío o es inválido"),
            HttpStatus.BAD_REQUEST
        );
    }

    @ExceptionHandler(Exception.class)
    public ResponseEntity<Map<String, Object>> manejarErrorGeneral(Exception ex) {
        Map<String, Object> response = construirRespuesta(HttpStatus.INTERNAL_SERVER_ERROR, "Ocurrió un error inesperado");
        response.put("detalle", ex.getMessage());
        return new ResponseEntity<>(response, HttpStatus.INTERNAL_SERVER_ERROR);
    }
}

AccesoService.java

La lógica de negocio de los 3 endpoints:

  • registrarEntrada: llama ms-reservas para validar estado CONFIRMADA → verifica que no exista otro acceso activo → crea Acceso → llama ms-espacios para marcar disponible=false.
  • registrarSalida: valida que la salida no esté ya registrada → calcula minutos → llama ms-espacios (disponible=true) → llama ms-reservas (finalizar) → guarda.
  • findByReserva: consulta simple por idReservaRef (para ms-pagos).
Service/AccesoService.java 
package estacionamientos.ms_accesos.Service;

import estacionamientos.ms_accesos.client.EspacioClient;
import estacionamientos.ms_accesos.client.ReservaClient;
import estacionamientos.ms_accesos.dto.AccesoCreateDTO;
import estacionamientos.ms_accesos.dto.AccesoResponseDTO;
import estacionamientos.ms_accesos.dto.ReservaResponseDTO;
import estacionamientos.ms_accesos.exception.ConflictException;
import estacionamientos.ms_accesos.exception.NotFoundException;
import estacionamientos.ms_accesos.model.Acceso;
import estacionamientos.ms_accesos.repository.AccesoRepository;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.time.LocalDateTime;
import java.time.temporal.ChronoUnit;

@Service
public class AccesoService {

    @Autowired private AccesoRepository accesoRepository;
    @Autowired private EspacioClient espacioClient;
    @Autowired private ReservaClient reservaClient;

    public AccesoResponseDTO registrarEntrada(AccesoCreateDTO dto) {
        ReservaResponseDTO reserva = reservaClient.findById(dto.getIdReserva());

        if (!"CONFIRMADA".equals(reserva.getEstado())) {
            throw new ConflictException("La reserva debe estar en estado CONFIRMADA para registrar entrada");
        }

        accesoRepository.findByIdReservaRef(dto.getIdReserva()).ifPresent(a -> {
            throw new ConflictException("Ya existe un acceso registrado para esta reserva");
        });

        Acceso acceso = new Acceso();
        acceso.setIdReservaRef(dto.getIdReserva());
        acceso.setIdEspacioRef(reserva.getIdEspacio());
        acceso.setHoraEntrada(LocalDateTime.now());
        acceso.setActivo(true);

        espacioClient.updateDisponibilidad(reserva.getIdEspacio(), false);

        return toDTO(accesoRepository.save(acceso));
    }

    public AccesoResponseDTO registrarSalida(Long id) {
        Acceso acceso = accesoRepository
            .findById(id)
            .orElseThrow(() -> new NotFoundException("Acceso no encontrado"));

        if (acceso.getHoraSalida() != null) {
            throw new ConflictException("La salida ya fue registrada para este acceso");
        }

        LocalDateTime horaSalida = LocalDateTime.now();
        long minutos = ChronoUnit.MINUTES.between(acceso.getHoraEntrada(), horaSalida);

        acceso.setHoraSalida(horaSalida);
        acceso.setMinutos((int) minutos);
        acceso.setActivo(false);

        espacioClient.updateDisponibilidad(acceso.getIdEspacioRef(), true);
        reservaClient.finalizar(acceso.getIdReservaRef());

        return toDTO(accesoRepository.save(acceso));
    }

    public AccesoResponseDTO findByReserva(Long idReserva) {
        Acceso acceso = accesoRepository
            .findByIdReservaRef(idReserva)
            .orElseThrow(() -> new NotFoundException("No existe acceso para la reserva " + idReserva));
        return toDTO(acceso);
    }

    private AccesoResponseDTO toDTO(Acceso a) {
        return new AccesoResponseDTO(
            a.getId(), a.getIdReservaRef(), a.getIdEspacioRef(),
            a.getHoraEntrada(), a.getHoraSalida(), a.getMinutos(), a.getActivo()
        );
    }
}

AccesoController.java

Controller/AccesoController.java 
package estacionamientos.ms_accesos.Controller;

import estacionamientos.ms_accesos.Service.AccesoService;
import estacionamientos.ms_accesos.dto.AccesoCreateDTO;
import estacionamientos.ms_accesos.dto.AccesoResponseDTO;
import jakarta.validation.Valid;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/accesos")
public class AccesoController {

    @Autowired private AccesoService accesoService;

    @PostMapping("/entrada")
    public ResponseEntity<AccesoResponseDTO> registrarEntrada(@Valid @RequestBody AccesoCreateDTO dto) {
        return ResponseEntity.status(HttpStatus.CREATED).body(accesoService.registrarEntrada(dto));
    }

    @PutMapping("/{id}/salida")
    public ResponseEntity<AccesoResponseDTO> registrarSalida(@PathVariable Long id) {
        return ResponseEntity.ok(accesoService.registrarSalida(id));
    }

    @GetMapping("/reserva/{idReserva}")
    public ResponseEntity<AccesoResponseDTO> findByReserva(@PathVariable Long idReserva) {
        return ResponseEntity.ok(accesoService.findByReserva(idReserva));
    }
}

Endpoints finales

POST /api/accesos/entrada Registra entrada física. Body: { "idReserva": 1 }
PUT /api/accesos/{id}/salida Registra salida. Calcula minutos y libera el espacio.
GET /api/accesos/reserva/{idReserva} Consulta acceso por reserva (lo usa ms-pagos).
Swagger disponible en http://localhost:8087/swagger-ui/index.html una vez arrancado el servicio.

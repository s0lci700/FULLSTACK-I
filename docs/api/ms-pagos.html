ms-pagos

localhost:8089 db_pagos /api/cobros /api/bancos /api/metodos-pago /api/tipo-tarjetas

Cobros, bancos, métodos de pago y tipos de tarjeta. Aplica fórmula de descuentos encadenados: tipo_cliente, suscripción y banco. Consulta ms-accesos, ms-tarifas y user-service vía Feign.

⚡ Feign: obtiene acceso de ms-accesos, tarifa vigente y multiplicador horario de ms-tarifas, vehículo + tipo de ms-vehiculos, espacio + tipo de ms-espacios, cliente de user-service.

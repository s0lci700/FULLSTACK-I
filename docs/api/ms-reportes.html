ms-reportes

localhost:8090 Sin BD propia /api/reportes

Servicio de reportes de solo lectura. No tiene base de datos propia — agrega datos de ms-espacios, ms-accesos y ms-pagos vía Feign en tiempo real.

⚡ Feign: lee de ms-espacios (ocupación), ms-accesos (por reserva) y ms-pagos (cobros por cliente).
ReporteController — /api/reportes
GET /api/reportes/ocupacion

Calcula la ocupación actual del estacionamiento consultando todos los espacios a ms-espacios y contando disponibles vs ocupados.

Response — 200 OK (OcupacionReporteDTO)
CampoTipoDescripción
totalEspaciosintTotal de espacios activos
disponiblesintEspacios con disponible=true
ocupadosinttotalEspacios − disponibles
Status
200 OK

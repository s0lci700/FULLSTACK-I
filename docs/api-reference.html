API Reference

Estacionamiento Inteligente · Spring Boot 3.5 · Java 21 · Base URL: http://localhost:8080

Autenticación requerida — Todos los endpoints excepto /api/auth/* requieren Authorization: Bearer {{token}}. Ejecuta el request #1 primero: el token se guarda automáticamente en la variable de colección token mediante el script de post-respuesta.

#55GET {{base}}/api/cobros/cliente/1200

#56POST {{base}}/api/cobros201
Genera el cobro para el acceso creado en #51. ms-pagos llama a 7 microservicios via Feign para aplicar la fórmula completa: precio × multiplicador horario × factor vehículo × factor espacio × (minutos/60) × descuentos.
Request Body
{
  "idAcceso": {{accesoId}},
  "idMetodoPago": 1,
  "idCliente": 1
}
#57POST {{base}}/api/cobros422
Intenta cobrar dos veces el mismo acceso. Espera 422 Business Rule Violation — la constraint UNIQUE en id_acceso_ref impide cobros duplicados.
Request Body (mismo que #56)
{
  "idAcceso": {{accesoId}},
  "idMetodoPago": 1,
  "idCliente": 1
}

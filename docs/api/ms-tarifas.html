ms-tarifas

localhost:8088 db_tarifas /api/tarifas /api/horarios-tarifa

Tarifas base y horarios con multiplicadores. ms-pagos consulta la tarifa vigente vía Feign para calcular el cobro. Solo una tarifa puede estar activa a la vez.

⚡ Feign: ms-pagos llama a GET /tarifas/vigente para obtener precioBaseHora.
Fórmula de cobro (ms-pagos):
monto_base = precioBaseHora × multiplicador_horario × factor_tipo_vehiculo × factor_tipo_espacio × (minutos / 60)
monto_final = monto_base × (1 − desc_tipo_cliente) × (1 − desc_suscripcion) × (1 − desc_banco)

ms-accesos

localhost:8087 db_accesos /api/accesos/**

Control de entrada y salida de vehículos. Al registrar entrada marca el espacio como no disponible y valida la reserva vía Feign. Al registrar salida calcula los minutos, libera el espacio y finaliza la reserva.

⚡ Feign: llama a ms-espacios (actualizar disponibilidad) y ms-reservas (validar y finalizar).
AccesoController — /api/accesos
POST /api/accesos/entrada

Registra la entrada de un vehículo. Valida que la reserva esté CONFIRMADA y que no exista otro acceso activo para esa reserva. Marca el espacio como no disponible.

PATCH /api/accesos/{id}/salida

Registra la salida del vehículo. Calcula minutos transcurridos, libera el espacio (disponible=true) y finaliza la reserva. Solo se puede llamar una vez por acceso.

ParamTipoDescripción

AccesoResponseDTO con fechaHoraSalida y minutos calculados. Estado: COMPLETADO.
